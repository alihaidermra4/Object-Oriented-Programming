# Object Oriented Programming (Theory + Lab)

**Institute of Management Sciences (IMSciences), Peshawar**
Course materials for the Object Oriented Programming course — theory concepts and lab tasks in C++, taught to BS students.

---

## 📘 About This Repository

This repository contains lecture content, code examples, and lab tasks for the OOP course. Content is organized around the four core pillars of Object-Oriented Programming — **Encapsulation, Abstraction, Inheritance, and Polymorphism** — so students can see not just *what* each topic is, but *why* it belongs where it does in OOP design.

> 📌 **Note:** Slides and lab tasks are updated periodically. Always check the latest version in this repo, especially for Exception Handling and STL content.

---

## 🗂️ Course Content Outline

### 0. Foundations
- Object-oriented design: history and advantages
- OOP terminology and features
- Classes and Objects
- Identification of classes and their relationships

### 1. Encapsulation
*Bundling data and behavior together, controlling access to them*
- Data encapsulation
- Attributes and methods of a class; identifying attributes/data members
- Constructors (and types of constructors)
- Destructors
- Access modifiers
- Passing objects to functions: by value vs. by reference
- `this` keyword in C++
- `const` vs non-`const` functions
- Static data members & functions

### 2. Polymorphism (Compile-Time)
- Function overloading
- Operator overloading

### 3. Class Relationships & Inheritance
- Composition ("has-a" relationship)
- Inheritance ("is-a" relationship)
- Multiple inheritance
- Protected access specifier in terms of inheritance
- Order of constructor calling in child and parent class

### 4. Polymorphism (Run-Time)
- Function overriding
- Polymorphism (general concept)

### 5. Abstraction
- Abstract classes and interfaces

### 6. Additional C++ Tools
- Standard Template Library (Vector, etc.)
- Exception Handling
